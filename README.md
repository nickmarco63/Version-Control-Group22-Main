The idea of this project is for the user to enter their name and their favorite word. Then the program will count the number of letters in that word. 
The program will also count which letter appears the most. If multiple letters repeat (the most) equally, then the first most commonly used letter will be listed. (Ex: racecar will return r)
Finally, the program greets the user, and then tells them the word with how many characters it contains. And the most used letter.
